- [x] OSMnx로 성수동 도로망 그래프 확보, 최단경로 탐색 검증 (`engine/network/`, 노드 464·엣지 1,396, 로컬 캐시)

- [ ] 결과 대시보드: 핵심 지표, 이탈원 차트, 시간대 차트
